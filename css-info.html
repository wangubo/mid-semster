CSS: Cascading Style Sheets fondly referred to as CSS is a simply designed language intended to simplify the process of making web pages presentable.
It is used to style our website.

css is divided into to inline css and external css
inline css is placedjust within the html tags or under the head tags but with selectors
external css is writen on a fresh page and linked to the html page . this link is put inthe head. the use of selectors forexample, the class and id selectors are comonly used
